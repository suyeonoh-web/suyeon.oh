카카오페이
분석 리서치 가이드

카카오페이는 단순 결제 앱이 아니에요. 결제 · 송금 · 청구서 · 투자 · 보험 · 인증까지 일상 금융을 통째로 다루는 ‘메신저 기반 종합 금융 플랫폼(슈퍼앱)’으로 어떻게 작동하고 어디로 가는가를 거꾸로 분석하는 프로젝트예요.

01 · 목적

리서치 목적

이번 리서치가 답해야 하는 핵심 질문

카카오페이는 사람들의 일상 금융을 통째로 책임지려는 슈퍼앱으로 성장 중이에요. “왜 결제·송금부터 시작했는가, 무료처럼 보이는데 어떻게 돈을 버는가, 카카오 안에서 어떤 역할인가, 앞으로 어디로 가는가”를 기존 서비스를 거꾸로 분석해 풀어내요.

핵심 5가지 질문

  1. 사람들은 왜 카카오페이를 쓰는가? (결제 / 송금 / 청구서 / 투자 중 어떤 이유로)
  2. 카카오톡 안에서 시작했는데, 왜 별도 앱으로 분리되었는가?
  3. 무료처럼 보이는 서비스로 카카오페이는 어떻게 돈을 버는가?
  4. 토스 · 네이버페이 대비 어떤 점이 다른가?
  5. 단순 결제 회사에서 종합 금융사로 어떻게 진화하고 있는가?
02 · 방향

리서치 핵심 방향

아래 3개의 축으로 동시에 진행해요.

축 1

서비스 이해

어떤 문제를 해결하는 종합 금융 플랫폼인가

축 2

시장 · 사업

어떻게 돈을 벌고, 핀테크 시장에서 어떤 위치인가

축 3

사용자 · 경쟁

사용자는 왜 카카오페이를 선택하는가

04 · 리서치 상세

각 주제별 무엇을 어떻게 조사할까?

6개 주제 각각 ① 목표 ② 핵심 질문 ③ 조사 항목 ④ 정리할 결과물 ⑤ 추천 자료·방법을 카드로 정리했어요. 모든 항목은 클릭해서 바로 수정할 수 있어요.

1

서비스 정체성과 핵심 기능군

카카오페이가 어떤 회사인지, 어떤 기능군으로 구성되어 있는지 명확히 정의해요.

🎯 핵심 질문
  1. 카카오페이는 어떤 회사인가? (한 줄로 정의하면)
  2. 핵심 기능은 무엇이며 어떻게 분류되는가?
  3. 어떤 기능이 사용자 수·매출에 가장 큰 비중인가?
  4. 어떤 기능이 미래 성장 동력인가?
📋 조사할 항목
  • 결제 — 오프라인 QR·바코드 / 온라인
  • 송금 — 카톡·계좌·해외
  • 청구서 — 받기·자동결제·정기납부
  • 인증 — 본인인증·간편인증서
  • 멤버십 — 적립·쿠폰·바코드
  • 투자 — 카카오페이증권 연계
  • 보험 — 미니보험·다이렉트보험
  • 대출·자산 — 대출비교·마이데이터·자산조회
📦 정리할 결과물
  • 카카오페이 전체 기능 지도 (마인드맵)
  • 핵심 vs 부가 기능 분류표
  • 한 문장 정체성 정의 (“카카오페이는 ___이다”)
📚 추천 자료·방법
  • 카카오페이 공식 사이트 메뉴 분석
  • 앱 직접 사용 (모든 메뉴 클릭해 보기)
  • IR 자료의 사업부문 설명

카카오페이가 속한 간편결제·핀테크 시장 전체를 이해해요.

🎯 핵심 질문
  1. 간편결제 시장 규모와 성장률은?
  2. 핀테크 시장의 주요 트렌드는?
  3. ‘슈퍼앱’ 트렌드란 무엇이고 왜 일어나는가?

  • 디지털금융 트렌드 (마이데이터·BNPL·오픈뱅킹)
  • 슈퍼앱 트렌드 (카카오·네이버·토스 비교)

📦 정리할 결과물

  • 3대 핵심 트렌드 정리

05 · 팀 운영 원칙

팀 운영 방식 (중요)

!

1명 = 1주제가 아니에요

카카오페이는 영역이 넓어요. 1명이 최소 2개 이상의 주제를 연결해서 조사해야 진짜 그림이 보여요. 수익 구조 · 사용자 · 기능 · 시장 · 경쟁이 서로 맞물려 있기 때문이에요.

왜 여러 주제를 함께 보는가?

· 수익 구조는 어떤 사용자가 어떤 기능을 쓰는지 봐야 보이고
· 경쟁 분석은 기능 비교 없이는 의미가 없으며
· 시장 흐름은 카카오의 슈퍼앱 전략과 연결되어야 해석돼요.

6명 역할 분담

각 카드 상단에 이름 · 진행 상태 · 마감일을 입력하고, 카드 아래 메모란에 자료 링크나 진행 메모를 자유롭게 적을 수 있어요.

1

팀원 1

서비스 정체성 + 사용자 사용 맥락
주제 1. 서비스 정체성주제 5. 사용 패턴
핵심 역할

“카카오페이는 무엇이고, 누가 어떻게 쓰는가” 분석

조사 포인트
  • 카카오페이의 핵심 정체성은 무엇인가 (결제 회사 / 송금 회사 / 종합 금융사)
  • 어떤 기능군이 가장 핵심인가 (결제·송금·청구서·투자·보험)
  • 사용자가 실제로 가장 자주 쓰는 기능은 무엇인가
  • 카카오톡 안의 페이 vs 별도 카카오페이 앱의 사용 차이
  • 사용자별 사용 패턴이 어떻게 다른가
반드시 정리해야 할 것
  • 카카오페이 기능 지도 (전체 기능 목록 정리)
  • 주요 사용 상황 5개 이상
  • 기존 금융앱 vs 카카오페이 비교
  • 사용자 입장의 정체성 정의 (“카카오페이는 ___이다”)
카카오페이는 결제 앱이 아니라 일상 금융을 통째로 다루는 슈퍼앱으로 진화 중이에요

07 · 사용자 리서치

실제 사용자는 어떻게 조사할까?

카카오페이는 기능이 많아서 사용자마다 쓰는 방식이 완전히 달라요. 인터뷰·설문에서 “어떤 기능을 쓰는지”부터 정확히 분류해야 의미 있는 결과가 나와요.

방법 1 · 사용자 인터뷰

사용 패턴이 다른 사람 5~8명을 1:1로 (결제 자주 쓰는 사람·송금형·청구서형·투자형 섞어서). 1명당 30분~1시간.

방법 2 · 설문 조사

구글폼·네이버폼으로 100명 이상. 주 사용 기능 · 사용 빈도 · 다른 앱 병행 여부를 숫자로 확인.

방법 3 · 후기·리뷰 분석

앱스토어·블로그·커뮤니티 후기 200개 이상을 모아 기능별로 분류. 자주 나오는 단어가 핵심 니즈와 불편한 점.

방법 4 · 직접 사용·관찰

팀원이 실제로 결제 · 송금 · 청구서 · 투자 · 보험 가입까지 직접 시도해 보고, 친구가 쓰는 것을 관찰.

인터뷰에서 꼭 물어볼 질문

  1. 카카오페이 어떤 기능을 가장 자주 쓰세요? (결제 / 송금 / 청구서 / 투자 / 인증 등)
  2. 처음 카카오페이를 쓰기 시작한 계기는 무엇이었나요?
  3. 토스나 네이버페이 같은 다른 앱과 어떻게 나눠 쓰시나요?
  4. 가장 편한 점과 가장 불편한 점은 무엇인가요?
  5. 카카오페이로 처음 해본 금융 행동이 있나요? (예: 첫 펀드 · 첫 미니보험)
08 · 사용자 페르소나

이 서비스는 누가 쓸까? (가상 사용자)

카카오페이는 한 명이 한 가지 방식으로만 쓰지 않아요. 주 사용 기능별로 4명을 그려보고, 리서치 후 실제 인물 묘사로 바꿔요.

P1

오프라인 결제 김대리

30대 직장인 · 편의점·카페·식당 결제 매일 사용
사용 상황

출근길 카페, 점심 식당, 편의점에서 QR·바코드로 결제. 카드 등록해서 자동 적립 챙김.

원하는 것 (니즈)

빠른 결제, 적립·할인 자동 적용, 영수증·지출 내역 자동 정리.

불편한 점

가맹점 따라 인식이 잘 안 됨, 매장 직원이 사용법을 잘 모르는 경우, 삼성페이가 더 편한 곳도 있음.

P2

공과금·청구서 박엄마

40대 맞벌이 · 가족 청구서 총무
사용 상황

관리비 · 통신비 · 보험료 · 학원비 등 매달 청구서를 카카오페이로 받고 결제. 가족 명의 청구서도 본인이 관리.

원하는 것 (니즈)

모든 청구서를 한 곳에서, 자동 결제, 가족 명의별 분류, 가계부 자동 정리.

불편한 점

청구서 안 보내는 기관 있음, 가족 카드와 분리하기 어려움, 지난 청구 내역 검색 불편.

P3

친구 송금·정산 이주임

20대 후반 직장인 · 친구·동료 송금 적극파
사용 상황

단톡방 회식 정산, 친구 모임 비용 분담, 경조사 송금 모두 카톡 안에서 처리.

원하는 것 (니즈)

계좌번호 안 묻고 보내기, 메시지 보내듯 자연스러운 흐름, 보낸 사람·안 보낸 사람 자동 확인.

불편한 점

카카오페이 안 가입한 친구는 송금 불가, 한도 제한, 자주 보는 친구일수록 정산 요청이 어색.

P4

투자·금융상품 최팀장

30대 후반 · 카카오페이증권·보험 이용
사용 상황

카카오페이증권으로 펀드·주식, 카카오페이에서 미니보험·대출 비교, 마이데이터로 자산 통합 관리.

원하는 것 (니즈)

한 앱에서 모든 금융 비교·관리, 카카오 데이터 기반 맞춤 추천, 작은 금액으로 가볍게 시작.

불편한 점

정보 깊이가 전문 금융 앱보다 얕음, 자산 통합 관리는 토스가 더 잘됨, 상품 선택지가 제한적.

09 · 시장 기회

카카오 밖, 전체 시장에서의 기회

카카오 안에서의 분석만으로는 부족해요. 전체 간편결제·핀테크 시장 규모 · 외부 사용자 행동 · 새로운 시장 기회 · 글로벌 사례를 함께 봐야 사업적 의미가 보여요.

1. 간편결제·핀테크 전체 시장 규모와 흐름

간편결제 시장 규모 (오프라인·온라인 비중), 1인당 결제·송금 횟수, 디지털금융 성장률. 카카오페이·토스·네이버페이 시장 점유율과 그 외 삼성페이·은행 앱 비중까지 함께 확인.

2. 카카오페이를 안 쓰는 사용자는 왜 다른 걸 쓸까

대안 서비스의 강점을 분석해서 카카오의 빈틈을 찾기.
· 토스: 더 단순한 UX, 카카오 친구 아니어도 송금 가능, 자산 통합 관리 강점
· 네이버페이: 네이버 쇼핑·웹툰 적립 강력
· 삼성페이·애플페이: 폰만 대면 결제, 가맹점 호환성
· 은행 앱: 자산 통합·예적금·대출 깊이

3. 새로운 시장 기회 (확장 가능성)

지금은 일상 결제·송금 중심인데, 아래 영역으로 넓힐 수 있어요.

  • · 자산관리·웰스테크 — 마이데이터 기반 통합 자산 관리·맞춤 추천
  • · 보험 디지털화 — 미니보험·다이렉트보험·맞춤 보험 추천
  • · 후불결제(BNPL) — 소액 후불결제, MZ 세대 타깃
  • · B2B 결제 솔루션 — 가맹점 경영 도구, 기업 비용 결제
  • · 글로벌 결제·송금 — 해외여행 결제, 유학생·외국인 노동자 송금
  • · 디지털 자산·증권 토큰화 — STO, 조각투자 같은 새 자산 영역

4. 글로벌·해외 시장 가능성

카카오톡은 국내 중심이지만, 결제·송금은 글로벌 시장이 훨씬 커요.
· 알리페이·위챗페이(중국) — 슈퍼앱 모델의 원형, 카카오페이가 따라가는 길
· 페이팔·Stripe(글로벌) — B2B·온라인 결제 강자
· Venmo·CashApp(미국) — 친구 송금 문화 만든 회사
· 동남아 진출 — 인도네시아·베트남 등 핀테크 성장기 시장

시장 분석에서 답해야 할 질문

  2. 카카오페이 외부 사용자는 왜 다른 서비스를 쓰는가?
